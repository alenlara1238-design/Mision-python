# Misión Python: Escape Lab

## Construye tu primer videojuego y descubre el poder de la programación

## Contexto

Bienvenido a **Misión Python: Escape Lab**.

En esta actividad desarrollarás desde cero un **Escape Room Digital** utilizando Python. A lo largo del ejercicio asumirás el papel de un personaje atrapado en un laboratorio secreto que deberá resolver acertijos, tomar decisiones y encontrar la forma de escapar.

Mientras construyes el proyecto descubrirás que la programación no consiste únicamente en realizar cálculos, sino en crear aplicaciones interactivas capaces de contar historias, resolver problemas y transformar ideas en software funcional.

Cada concepto aparecerá en el momento en que el proyecto lo necesite, permitiéndote comprender para qué sirve cada herramienta del lenguaje mientras desarrollas un videojuego completamente funcional.

Al finalizar la actividad habrás construido tu primer proyecto completo en Python.

---

# Objetivo General

Desarrollar un videojuego interactivo en Python que permita comprender cómo se construye una aplicación real utilizando los fundamentos de la programación.

---

# Objetivos Específicos

Al finalizar esta actividad serás capaz de:

- Comprender el flujo de ejecución de un programa en Python.
- Utilizar variables para almacenar información.
- Interactuar con el usuario mediante entradas y salidas de datos.
- Implementar estructuras de decisión para controlar el comportamiento del juego.
- Utilizar ciclos para mantener la interacción con el jugador.
- Organizar el código mediante funciones.
- Aplicar buenas prácticas básicas de organización y legibilidad del código.
- Comprender cómo pequeños bloques de código pueden combinarse para crear una aplicación completa.

---

# Nivel Requerido

Esta actividad está dirigida a estudiantes que se encuentran dando sus primeros pasos en Python.

No es necesario conocer Programación Orientada a Objetos, estructuras de datos avanzadas ni bases de datos.

---

# Conocimientos Previos

Antes de comenzar debes ser capaz de:

- Tener instalado Python y Visual Studio Code.
- Ejecutar programas en Python.
- Utilizar `print()`.
- Utilizar `input()`.
- Declarar variables sencillas.

---

# Desarrollo de la Actividad

La construcción del videojuego se realizará de forma incremental.

## Primera etapa

Construirás la base del juego.

En esta parte desarrollarás:

- Pantalla de bienvenida.
- Registro del jugador.
- Menú principal.
- Primeras decisiones.
- Rutas del juego.
- Condiciones de victoria y derrota.

---

## Segunda etapa

Agregarás nuevas funcionalidades al videojuego.

Entre ellas:

- Inventario.
- Habitaciones.
- Llaves.
- Contraseñas.
- Acertijos.
- Múltiples finales.

---

## Tercera etapa

Personalizarás el proyecto agregando tus propias ideas.

Podrás incorporar:

- Nuevas habitaciones.
- Objetos.
- Personajes.
- Acertijos.
- Finales alternativos.
- Cualquier otra característica que haga tu juego más interesante.

---

# Producto Final

Al finalizar la actividad habrás creado un videojuego funcional que incluya como mínimo:

- Una historia interactiva.
- Navegación mediante decisiones.
- Inventario básico.
- Múltiples caminos posibles.
- Condiciones de victoria y derrota.
- Código organizado mediante funciones.
- Un programa claro, ordenado y fácil de leer.
