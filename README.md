# Misión Python: Escape Lab

## Bienvenido

Has despertado dentro de un laboratorio secreto.

No recuerdas cómo llegaste hasta allí.

Las luces parpadean, las puertas están bloqueadas y una voz proveniente de los altavoces anuncia:

> **"Protocolo de confinamiento activado."**

Tu única misión será explorar las instalaciones, resolver acertijos, encontrar objetos útiles y descubrir la forma de escapar antes de que el laboratorio cierre definitivamente.

Pero no estarás solo.

Durante esta aventura irás aprendiendo cómo un desarrollador construye un programa real utilizando Python.

Cada nueva característica del videojuego aparecerá cuando el proyecto la necesite, igual que sucede durante el desarrollo de software en el mundo profesional.

---

# Objetivo del proyecto

Construir paso a paso un videojuego interactivo en consola mientras se aprenden los fundamentos de Python.

Al finalizar el proyecto habrás desarrollado un programa que utiliza:

- Variables
- Entrada y salida de datos
- Conversión de tipos
- Condicionales
- Operadores relacionales y lógicos
- Ciclos `while`
- Listas
- Funciones
- Organización del código en módulos

---

# Historia

El laboratorio pertenecía a una empresa dedicada al desarrollo de inteligencia artificial.

Algo salió mal.

Todos los investigadores desaparecieron.

Las puertas de seguridad fueron bloqueadas automáticamente.

Solo queda un registro de emergencia que indica que existe una salida...

...pero para abrirla será necesario recorrer distintas habitaciones, encontrar objetos, resolver desafíos y descubrir el código secreto del laboratorio.

¿Podrás escapar?

---

# Reglas del juego

Durante la aventura podrás:

- Explorar habitaciones.
- Recoger objetos.
- Consultar tu inventario.
- Resolver acertijos.
- Utilizar objetos encontrados.
- Descubrir pistas.
- Desbloquear nuevas zonas.
- Escapar del laboratorio.

Algunas decisiones tendrán consecuencias.

Explorar con atención será la diferencia entre escapar... o quedar atrapado para siempre.


---

# Cómo ejecutar el proyecto

Desde la terminal:

```bash
python main.py
```

o

```bash
python3 main.py
```

dependiendo del sistema operativo.

---

# Objetivo educativo

Este proyecto fue diseñado siguiendo una metodología de construcción incremental.

No aprenderás los conceptos de Python de manera aislada.

Cada nuevo concepto aparecerá cuando el videojuego lo necesite.

De esta forma comprenderás cómo los desarrolladores utilizan los fundamentos de programación para construir aplicaciones reales.
